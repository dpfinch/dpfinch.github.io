Links

Here are some interesting websites - some academic some friends and some just cool stuff:

School of Geosciences

    I am based at the School of Geosciences at the University of Edinburgh and this website will give you load of information about the diverse and successful school. More information can be found here: School of Geosciences.

Palmer Group

    The Palmer Group is a research group at The University of Edinburgh that I work for. It is headed by Prof. Paul Palmer and has a number of PhD students and post-docs doing reasearch on a diverse number of topics from atmospheric chemistry to modelling human migration to using Earth System models to help in the search for extra-terrestrial life.More information can be found here: Palmer Group.

BORTAS

    Here you can find information about the BOReal forest fires on Tropospheric oxidants over the Atlantic using Aircraft and Satellites (or BORTAS for short). This is a collaborative field campaign looking at atmospheric chemistry downwind from forest fires. Most of my research is based on data collected during this campaign in 2011.More information can be found here: BORTAS.

GEOS-Chem

    Information about the GEOS-Chem chemistry transport model that I use can be found here. This website gives information about the model as well as the community that created, maintain and use GEOS-Chem from all over the world. More information can be found here: GEOS-Chem.

Code Academy

    This is a great website that I've found really useful in learning coding. It teaches a number of different languages from beginner level. There are all sorts of projects to try out as well - so it seems like you are creating something, not just going through the motions. More information can be found here: Code Academy.

MRes CLimate and Atmospheric Science

    This is a link to a modified version of the masters course I did. Its an MRes (Masters in Research) so its a lot more focused on your own project and developing as an independant researcher than other masters programs. I really liked this course and I wouldn't be doing a PhD today without it. More information can be found here: MRes Climate and Atmospheric Science.

Dizem Bella

    Dizem Bella is a nutritional rehabilitation centre and school in Karaga, northern Ghana. This website gives information about a charity campaign set up by some friends of mine whose ultimate goal is to make the centre completely self-sufficient. Its a little charity that is a great example of how a simple idea can have a massive effect on peoples lives. More information can be found here: Dizem Bella.

Crew Cakes

    This is a blog set up by some friends of mine in the office after a demand by everyone for everyone else's cake recipes. Named after the building we are in (Crew Building) the Crew Cake Crew blog features many of the cakes we make for each other for various occasions. Along with the recipe there is usually some fun facts about the cake/ingredients. More tastyness can be found here: Crew Cakes.
